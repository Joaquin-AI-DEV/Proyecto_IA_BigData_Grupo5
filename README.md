# Proyecto_IA_BigData_Grupo5

Nombre del proyecto:
StockPulse

Objetivo del proyecto:
El objetivo es desarrollar un modelo de predicción de ventas que analice datos históricos o simule datos nuevos para ayudar con la toma de decisiones de compra y gestión de inventario. 

Miembros:
    - Sergio Romero: Data Engineer (ingesta, limpieza, EDA). 
    - Deninson Tapia: Data Analyst Engineer (modelado, evaluación). 
    - Joaquín Fernández: Platform (API, despliegue, orquestación).
    - Jimmy Chen: BI (dashboard, KPIs).

Organización del repo: 

El repositorio se organizará según las siguientes carpetas, con el objetivo de mantener el proyecto ordenado:
    <pre> ```bash repo/ ├── docs/ │ └── Fase0_Presentacion_y_Viabilidad.pdf ├── src/ ├── data/ ├── environment/ ├── README.md └── .gitignore ``` </pre>

📁 docs/
Contendrá toda la documentación del proyecto:
    - Entregables en PDF
    - Presentaciones
    - Diagramas
    - Memoria técnica

📁 src/
Contendrá el código fuente del proyecto:
    - Scripts de procesamiento de datos
    - Modelos de Machine Learning
    - Scripts de entrenamiento
    - Evaluación del modelo
    - Posible API o interfaz futura

📁 data/
Contendrá los datos utilizados en el proyecto:
    - Datos brutos (raw)
    - Datos procesados
    - Datos de prueba

⚠️ No se subirán datasets pesados al repositorio.
Se incluirán solo muestras o instrucciones de descarga.

📁 environment/
Contendrá la configuración del entorno:
    - requirements.txt
    - environment.yml
    - Configuración de dependencias
    - Notebooks de pruebas iniciales
